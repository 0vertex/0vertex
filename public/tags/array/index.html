What is a property wrapper? Swift 5.1 added a new feature called PropertyWrapper, a clean and easy way of abstracting core logic behind any stored property. You might have already seen a few of these like @State @Binding @Pubished etc. Property Wrapper provides the layer of separation for the business logic of the property and its definition.
